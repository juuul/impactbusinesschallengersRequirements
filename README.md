# Impact Business Challengers

A website for Impact Business Challengers. This repository contains the source code and requirements for the website.

## Project Structure

```
.
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Stylesheet
├── js/
│   └── main.js        # JavaScript functionality
├── assets/            # Images, fonts, and other assets
├── requirements/      # Website requirements and content documentation
└── README.md          # Project documentation
```

## Getting Started

### Prerequisites

No build tools required - this is a simple HTML/CSS/JavaScript website.

### Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/juuul/impactbusinesschallengersRequirements.git
   cd impactbusinesschallengersRequirements
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local web server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     ```

3. Navigate to `http://localhost:8000` in your browser
