# Requirements Documentation

This folder contains organized requirements and content for the Impact Business Challengers website.

## Files

1. **1-brand-guidelines.md** - Brand colors, typography, and logo guidelines
2. **2-website-requirements.md** - Technical requirements, scope, and sitemap
3. **3-home-page-content.md** - Content for the home page
4. **4-services-content.md** - Content for all service pages (Consultancy, Detachering, Business Challenges)
5. **5-scans-content.md** - Content for the free scans pages (180° and 360°)
6. **6-about-us-content.md** - Team member bios and core values
7. **7-cases-structure.md** - Template structure for case studies
8. **8-contact-content.md** - Contact page content and form field requirements

## Original Document

The original requirements document (Impact_Website Requirements.docx) is also stored in this folder for reference.

## Using These Requirements

Each markdown file contains the content and specifications needed to build the corresponding section of the website. Developers should refer to these files when implementing the website pages.
